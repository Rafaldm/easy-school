Plataforma interativa para aprender PHP com sistema de metas diárias, calendário de progresso e ranking pessoal.

## ✨ Funcionalidades

- ✅ 20 exercícios básicos de PHP com explicações detalhadas
- ⏱️ Cronômetro de tempo de estudo
- 🎯 Sistema de metas personalizável (finalizações, acerto %, tempo)
- 📅 Calendário visual de progresso mensal
- 🏆 Ranking com os 5 melhores tempos
- 💾 Dados salvos localmente (LocalStorage)

## 🚀 Como Usar

1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/treinamento-php.git
```

2. Ou Abra em https://rafaldm.github.io/easy-school
   

4. Configure suas metas clicando no ícone ⚙️

5. Comece a praticar!

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript
- Bootstrap 5.3.2
- LocalStorage API

## ⚙️ Configurações de Metas

Personalize suas metas diárias:

- **Finalizações**: 1-10 (padrão: 2)
- **Taxa de Acerto**: 50-100% (padrão: 70%)
- **Tempo Mínimo**: 5-120 min (padrão: 20 min)
